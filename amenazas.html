Amenazas Comunes en Ciberseguridad

Phishing

Intento de engañar a usuarios para obtener información confidencial simulando ser una entidad confiable.

Ransomware

Tipo de malware que cifra los archivos de un sistema y exige un rescate para liberarlos.

Malware

Software malicioso diseñado para causar daño, infiltración o robo de datos sin consentimiento.

Ataques DDoS

Ataques que saturan un sistema o red con tráfico excesivo, provocando que deje de funcionar correctamente.

Casos Reales

Un mensaje de texto falso de "Correos de Chile": solicitó a miles de clientes el pago de Aduanas de paquetes retenidos con el fin de que hicieran click en el enlace enviado para poder robar sus credenciales de acceso.

Este ransomware se propagó rápidamente a través de una vulnerabilidad en el protocolo SMB de Windows, afectando a miles de computadoras en todo el mundo y paralizando sistemas en hospitales, empresas y organizaciones gubernamentales.

Este ataque, que utilizó la botnet Mirai, afectó a servicios como Twitter, Spotify, Netflix, Amazon y GitHub, entre otros.
